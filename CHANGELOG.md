# CHANGELOG — Despacho Náutico Legal

Todos los cambios notables en este proyecto serán documentados en este archivo.

---

## [1.0.0] — 2025 (Versión Inicial)

### Añadido
- Diseño completo one-page con 8 secciones funcionales
- Navbar fijo con efecto glassmorphism al hacer scroll
- Menú hamburguesa para mobile
- Hero con imagen de fondo real de marina, orbs animados y efecto grain
- Sección de 8 tarjetas de servicios con imágenes de Unsplash y hover effects
- Mini-stats flotantes en el hero (1,200+ trámites, 12+ años, 98% éxito)
- Barra de confianza con iconos de servicios principales
- Sección de experiencia con layout imagen + texto + estadísticas
- Sistema de tabs para trámites detallados (8 categorías, 30+ servicios específicos)
- Grid de 8 beneficios con animación de barra inferior en hover
- Sección proceso de 4 pasos con línea conectora animada
- Formulario de contacto con validación JS en tiempo real
- Toast de confirmación de envío de formulario
- Footer con 4 columnas, links de navegación, contacto y redes sociales
- Animaciones reveal on scroll con IntersectionObserver
- Diseño completamente responsive (desktop, tablet, mobile)
- Modo oscuro por defecto con paleta navy + blue + cyan

### Diseño
- Tipografía: Playfair Display (display) + Outfit (body)
- Sistema de variables CSS para tokens de diseño
- Gradientes tipo startup (Stripe/Vercel inspired)
- Efecto grain/noise sutil en overlay
- Orbs flotantes animados en hero

### Técnico
- Archivo único autocontenido (index.html)
- Sin dependencias npm, 100% CDN
- Compatible con GitHub Pages sin configuración adicional
- JavaScript vanilla, sin frameworks
