# Despacho Náutico Legal — Sitio Web Oficial

Sitio web profesional de una empresa de gestión de trámites administrativos náuticos en México.

## Descripción
Landing page de una sola página (one-page) completamente estática, diseñada para presentar los servicios de gestión marítima ante SEMAR, CONANP, RPMN y Capitanías de Puerto.

## Tecnologías
- HTML5 semántico
- TailwindCSS (via CDN)
- JavaScript vanilla
- Google Fonts (Playfair Display + Outfit)
- Font Awesome 6.5 (iconos)
- Imágenes: Unsplash (alta calidad, libres de uso)

## Estructura de Secciones
1. **Hero** — CTA principal con imagen de marina
2. **Servicios Principales** — 8 tarjetas con imagen real
3. **Experiencia** — Sección panorámica con estadísticas
4. **Trámites Detallados** — Panel de tabs con todos los servicios específicos
5. **Beneficios** — 8 tarjetas de ventajas competitivas
6. **Proceso** — 4 pasos del flujo de trabajo
7. **Formulario de Contacto** — Validación JS + toast de confirmación
8. **Footer** — Links, contacto y redes sociales

## Cómo Publicar en GitHub Pages
1. Sube todos los archivos a un repositorio público en GitHub
2. Ve a Settings → Pages
3. Source: Deploy from branch → main → / (root)
4. En unos minutos estará en: `https://usuario.github.io/repositorio/`

## Personalización
- Reemplaza los datos de contacto en la sección `#contacto` y el footer
- Cambia el nombre/logo en el navbar y footer
- Actualiza las imágenes de Unsplash con URLs propias si se desea
- Ajusta colores en las variables CSS `:root` al inicio del `<style>`
